# Sentiment Annotation Guidelines

## Objetivo

Classificar reviews como Positivo ou Negativo.

## Definição de Positivo
Texto expressa satisfação, recomendação ou emoção positiva predominante.

## Definição de Negativo
Texto expressa reclamação, frustação ou emoção negativa predominante.

## Regra de Sentimento Predominante
Se houver mistura, considerar o sentimento mais forte.

## Regra para Ironia
Se houver ironia clara com com crítica implícita, classificar como Negativo.

## Casos Ambíguos
Quando houver dúvida, registrar justificativa na planilha
